# Educase / PopX Frontend Assignment

## Project Overview

This project is a responsive frontend web application developed as part of the Educase assignment. The application simulates a simple user interface for a platform called **PopX**, where users can create an account, log in, and view their profile information.

The project demonstrates frontend development skills using **React, Vite, and modern UI design practices**.

## Live Demo

🔗 https://educase-task-indol-alpha.vercel.app

## Features

* Welcome screen with navigation options
* Create account (registration) form
* Login page UI
* Profile page layout
* Responsive mobile-style design
* Client-side form validation
* Navigation using React Router

## Technologies Used

* React.js
* Vite
* JavaScript (ES6+)
* HTML5
* CSS3
* React Router DOM

## Installation and Setup

Clone the repository:

```bash
git clone https://github.com/Saifurrahman21/educase_task.git
```

Go to the project directory:

```bash
cd educase_task
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:5173
```

## Deployment

The project is deployed on **Vercel** for fast and reliable hosting.
